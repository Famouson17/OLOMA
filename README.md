# Oloma Language Web Page

This project is a simple, single-page HTML document created to present information about the **Oloma language**, spoken in Edo State, Nigeria.

---

## 🌍 About

The purpose of this page is to:

* Highlight the existence of the Oloma language
* Provide basic information in a simple web format
* Contribute to the online presence of the Oloma-speaking community
